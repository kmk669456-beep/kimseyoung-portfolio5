소개

학력 한남대학교 회계학과·빅데이터응용학과 다전공 4학년 재학 중 (2027년 2월 졸업 예정)
소개 세무법인 인턴으로 자료 정리, 수치 검토, 전자신고 보조 등 회계·사무 실무를 경험했습니다. 회계학과·빅데이터 다전공과 민관데이터 프로젝트 경험을 바탕으로, 데이터를 활용한 체계적 업무 수행 역량을 갖추고 있습니다.

좌우명

실패가 지닌 가치를 인정한다. 나는 실패를 그냥 털어버리지 않고 실패가 지닌 중요한 가치를 추출해낸다.

강점

  • 업무를 미루지 않고 제때 처리하는 신속한 실행력
  • 체계적인 성격과 일관된 업무 처리 방식
  • 회계학적 지식과 데이터 처리 능력
  • 피드백 수용력이 높고, 실패를 학습으로 전환하는 태도

학습내용

회계 · 세무

학습 배경

고등학교 시절 회계를 처음 접한 이후, 단순 암기가 아닌 기업 활동을 설명하는 언어라는 점에 흥미를 느껴 관련 과목과 자격 과정을 통해 체계적으로 학습해왔습니다.

핵심 학습 영역

  • 재무회계: 자산·부채·자본 구조 이해, 재무제표 작성 및 분석
  • 중급회계: 투자자산, 유형·무형자산, 부채·자본 회계처리 판단
  • 원가회계: 요소별·부문별·제품별 원가 계산 구조 이해
  • 재무관리: 자본조달, 현금흐름, 자본구조·레버리지 분석
  • 세무회계·세법: 법인세·부가가치세 구조 및 세무조정 흐름
  • 회계감사: 감사 절차 이해 및 통계·EDA 기법 활용

학습 방식

개별 과목 암기보다는 재무 흐름과 의사결정 구조를 중심으로 학습하며, 재무제표 수치가 왜 그렇게 나타나는지를 설명할 수 있도록 정리했습니다.

빅데이터 · AI

학습 계기

AI 환경 변화 속에서 회계·세무 지식을 데이터 기반으로 확장할 필요성을 느껴 빅데이터 분석학과를 다전공으로 선택했습니다.

핵심 학습 영역

  • 통계: 확률 변수, 확률분포 이해 및 기초 추론
  • Python: 데이터 수집·변환·분석, 알고리즘 사고, 간단한 웹 구현
  • R: 데이터 처리·분석·시각화 중심 통계 분석
  • EDA: 데이터에서 의미를 도출하고 구조화하는 탐색 과정
  • 머신러닝: SVM, 앙상블, 그래프 기반 모델 이론 및 실습
  • 영상처리: 이미지·영상 데이터 기반 정보 추출

학습을 통해 얻은 관점

언어 자체보다 문제를 어떻게 구조화하고 접근하는지가 중요하다는 점을 체감했으며, AI와 데이터는 결과 생성 도구가 아닌 판단 근거를 보조하는 도구로 활용하는 방향을 지향합니다.
